### **`Task 2: List Comprehension with Condition`**

**`Problem:`**

1. `Given a list of numbers from 1 to 10, use a list comprehension to create a new list containing only the even numbers.`  
   2. `Print the new list.`

**`Example Output:`**

### **`Task 3: Dictionary Comprehension`**

**`Problem:`**

1. `Create a dictionary comprehension where the keys are numbers from 1 to 5, and the values are their squares.`  
   2. `Print the resulting dictionary.`

**`Example Output:`**

`{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}`

### **`Task 4: Nested Comprehension`**

**`Problem:`**

1. `Given a list of lists [[1, 2, 3], [4, 5], [6, 7]], use a nested list comprehension to flatten the list into a single list.`  
   2. `Print the flattened list.`

**`Example Output:`**

`[1, 2, 3, 4, 5, 6, 7]`

### **`Task 5: Set Comprehension with Condition`**

**`Problem:`**

1. `Given a list of numbers, use a set comprehension to create a set containing only the unique squares of the even numbers.`  
   2. `Print the resulting set.`

**`Example Output:`**
